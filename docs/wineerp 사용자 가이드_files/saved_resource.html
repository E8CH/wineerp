시작하기 — 로그인과 역할

앱을 열면 로그인 화면이 먼저 뜹니다. 로그인 후에는 하단에 6개의 탭이 나타납니다.

로그인

  1. 이메일과 비밀번호를 입력합니다.
  2. 로그인 버튼을 누릅니다. 아이디·비밀번호가 틀리면 “이메일 또는 비밀번호가 올바르지 않습니다.”가 뜹니다.
  3. 로그인에 성공하면 스캔 화면으로 바로 넘어갑니다.
앱을 완전히 종료하면 다시 로그인해야 합니다. 로그인 정보는 앱을 켜 둔 동안에만 유지됩니다. (별도의 로그아웃 버튼은 없습니다.)

두 가지 역할 — 직원과 관리자

계정에는 역할이 있고, 역할에 따라 일부 기능이 제한됩니다. 화면 구성(6개 탭)은 누구에게나 똑같이 보이지만, 관리자 전용 기능은 열었을 때 잠금 안내가 뜨거나 버튼이 보이지 않습니다.

  • 직원 매일의 핵심 작업 — 스캔·등록·입고, 재고·내역 확인, 입고 수량 수정까지 할 수 있습니다.
  • 관리자 직원이 할 수 있는 모든 것에 더해 — 입고 취소, 리포트, 모델 수정·삭제, 활동 로그를 볼 수 있습니다.
1

핵심 흐름 — 스캔 → 확인 → 입고

매일 가장 많이 쓰는 흐름입니다. 병의 바코드를 찍으면 앱이 알아서 다음 할 일을 안내합니다.

바코드 스캔

  1. 바코드를 프레임 안에 비춥니다. 화면 안내: “바코드를 프레임 안에 비추세요”. 옆 병을 잘못 읽지 않도록, 같은 코드가 연속으로 안정 인식돼야 확정됩니다.
  2. 인식되면 화면이 초록으로 멈추고 그 장면을 붙잡습니다. “인식됨 · 아래에서 확인하세요”. 다시 찍으려면 다시 스캔을 누릅니다.

인식 뒤에는 그 바코드가 이미 등록된 와인인지에 따라 화면이 달라집니다.

이미 등록된 와인이면

  • 딱 하나로 매칭되면 바로 입고 확인 카드가 뜹니다(생산자·모델명·빈티지·현재고).
  • 같은 바코드에 후보가 여럿이면 “라벨을 보고 골라주세요”가 뜹니다. 실제 라벨과 대조해 직접 선택하세요.
  • 찾는 게 없으면 목록 끝의 찾는 빈티지가 없어요로 새 등록으로 갈 수 있습니다.

처음 보는 와인이면

미등록 와인 카드가 뜹니다. 새로 등록을 눌러 라벨 사진으로 새 모델을 만듭니다. (아래 신규 등록 참고)

신규 등록 직원

  1. 라벨 사진 촬영 (필수) 사진을 찍어야 나머지 입력 칸이 나타납니다. 라벨이 곧 근거입니다.
  2. 모델명·생산자·빈티지를 채웁니다. 모델검색을 누르면 사진에서 자동으로 초안을 채워 줍니다(“AI 추론” 표시). 라벨과 다르면 반드시 고치세요 — 자동값은 그대로 저장되지 않고, 확인해야 등록됩니다. 대신 직접입력으로 손수 입력해도 됩니다.
  3. 빈티지는 연도 4자리를 넣거나, 논빈티지면 NV를 켭니다.
  4. 등록하고 계속을 누릅니다. 바코드는 방금 스캔한 값이 자동으로 연결됩니다.
지역·국가·품종은 등록 화면에 없습니다. 등록을 마친 뒤 모델 탭 › 수정에서 채웁니다(관리자).

입고 수량 넣고 완료 직원

  1. 수량을 조절합니다(＋/－, 최소 1병).
  2. 필요하면 메모 추가로 메모(선택)를 남깁니다. “파손·명세서 불일치 등”. 최대 500자.
  3. 완료를 누릅니다. “입고 완료 · (모델명)” 안내가 뜨고, 다음 병을 위해 스캔이 준비됩니다.
버튼을 연달아 눌러도 같은 입고가 중복 저장되지 않습니다. 안심하고 눌러도 됩니다.
2

초기 세팅 모드 — 보유 재고 한 번에 넣기

처음 도입할 때, 이미 매장에 있는 와인들을 빠르게 등록하는 전용 모드입니다. 등록에 집중하는 흐름이라 낱개 입고와 다릅니다.

  1. 스캔 화면 오른쪽 위 초기 세팅 모드 아이콘을 눌러 들어갑니다. 위쪽에 “초기 세팅 모드” 배너와 등록 종수 카운터가 뜹니다.
  2. 병을 계속 스캔하며 등록합니다. 등록 폼 제목이 보유 와인 등록으로 바뀝니다. 보유 수량 (선택)에 지금 갖고 있는 병 수를 넣으면 기준 재고로 잡힙니다. 비워두면 모델만 만들어집니다.
  3. 각 병은 등록하고 다음 병으로 빠르게 이어갑니다. 다 되면 배너의 나가기.
세팅 모드의 등록은 낱개 “입고” 기록을 만들지 않습니다(보유 수량은 기준 재고로 들어갑니다). 이미 등록된 병을 다시 찍으면 이미 등록된 와인으로 알려 줍니다.
3

재고 탭 — 지금 몇 병?

빈티지별 현재고를 한눈에 봅니다. 누구나 볼 수 있고, 여기서는 수정 없이 조회만 합니다.

  • 각 줄은 빈티지 단위입니다 — 라벨 썸네일, 모델명, 생산자·빈티지, 오른쪽에 현재고 배지.
  • 줄을 누르면 큰 라벨 사진과 함께 지역·국가·품종·현재고 상세가 열립니다.
  • 재고가 0인 와인도 사라지지 않고 흐리게 남습니다.
  • 오른쪽 위 새로고침으로 최신값을 다시 불러올 수 있습니다.
4

내역 탭 — 입고 기록과 정정

언제·누가·몇 병 넣었는지 카드로 봅니다. 잘못 넣었으면 여기서 바로잡습니다.

보기

  • 위쪽에서 일간 / 주간 / 월간 기간을 고릅니다(기본 일간).
  • 각 카드: 라벨 썸네일, 모델명, 빈티지 · 날짜·시각 · 담당자, 수량. 상황에 따라 배지가 붙습니다 — 수정됨, 초기 세팅, 메모 미리보기.
  • 당겨서 새로고침으로 갱신됩니다.

수정·취소 — 카드를 누르면

  • 수량 수정 직원 — 수량을 바꾸고 수정 사유 (선택)를 적은 뒤 저장. 큰 라벨 사진도 함께 보입니다.
  • 입고 취소 관리자 — 입고 취소 버튼은 관리자에게만 보입니다. 취소는 재고에서 통째로 빼는 되돌리기 어려운 작업이라 권한을 달리 둡니다.
수정하거나 취소하면 재고와 리포트에도 곧바로 반영됩니다. 모든 변경은 활동 로그에 남습니다.
5

리포트 탭 관리자

기간별 입고 요약과 상위 품목을 보고, 엑셀로 내려받습니다. 관리자만 열 수 있습니다.

  • 직원이 열면 “리포트는 관리자만 볼 수 있습니다” 잠금 화면이 뜹니다.
  • 기간은 주간 / 월간 중에서 고릅니다.
  • 맨 위 요약 3종 — 총 입고 n병 · 기록 n건 · 품목 n종 — 아래 일별 막대그래프와 상위 품목 목록.
  • 오른쪽 위 엑셀 다운로드 아이콘으로 지금 기간의 데이터를 xlsx로 내보내 공유할 수 있습니다.
6

모델 탭 — 등록된 와인 관리

등록된 모델(제품)을 찾고, 정보를 보완하고, 필요 없어진 모델을 정리합니다.

찾기 직원

  • 모델명·생산자·지역 검색 칸에 입력하면 즉시 걸러집니다.
  • 등록일 버튼으로 등록일 범위를 지정해 좁힐 수 있습니다.
  • 카드에는 지역·국가·품종, 빈티지 종수 · 재고 · 등록일이 함께 보입니다. 카드를 누르면 상세(큰 사진·빈티지별 재고).

수정·삭제 관리자

상세를 열면 관리자에게만 수정·삭제 버튼이 보입니다.

  • 수정 — 생산자·모델명(필수)·지역·국가·품종을 채웁니다. 여기서 넣은 값은 재고·입고 내역에도 바로 반영됩니다.
  • 삭제 — 확인창을 거치는 “아카이브”입니다. 삭제하면 카탈로그·재고·리포트에서 사라지고, 관련 입고 기록도 내역에서 빠집니다. 단 원장 데이터는 보존되며, 같은 바코드로 다시 등록할 수 있습니다. 삭제 사실은 활동 로그에 “모델 삭제”로 남습니다.
7

활동 로그 탭 관리자

누가·언제·무엇을 넣고·고치고·지웠는지 시간순으로 봅니다. 관리자만 볼 수 있습니다.

  • 카드가 아니라 연속 리스트입니다 — 변경 한 건이 한 줄. 각 줄에 요약 · 작업자 · 시각.
  • 줄을 누르면 상세가 열립니다 — 수량 변경은 “10병 → 15병”처럼, 모델 수정은 바뀐 항목만 before → after로 보여줍니다.

기록되는 활동

  • 입고 · 수량 수정 · 입고 취소
  • 모델 등록 · 모델 수정 · 모델 삭제 · 초기재고
방금 한 작업이 안 보이면 다른 탭에 갔다가 로그 탭으로 다시 들어오거나, 오른쪽 위 새로고침을 누르세요.

자주 겪는 상황

카메라가 안 켜져요

“카메라를 사용할 수 없습니다” 안내가 뜨면, 휴대폰 설정에서 카메라 권한을 허용한 뒤 다시 시도하세요.

“로그인이 만료되었습니다”

일정 시간이 지나면 다시 로그인이 필요합니다. 로그인 화면에서 다시 로그인하면 하던 입력값은 대개 유지됩니다.

같은 바코드에 후보가 여러 개예요

자동으로 고르지 않습니다. 실제 라벨을 보고 직접 선택하세요. 없으면 찾는 빈티지가 없어요로 새로 등록합니다.

잘못 입고했어요

내역 탭에서 해당 카드를 눌러 수량을 수정(직원 가능)하거나, 입고 취소(관리자)를 하세요.

지역·국가·품종을 넣고 싶어요

등록 화면에는 없습니다. 모델 탭 › 상세 › 수정(관리자)에서 채웁니다.

화면 값이 오래된 것 같아요

탭을 다시 눌러 들어오면 최신으로 갱신됩니다. 각 화면의 새로고침이나 당겨서 새로고침도 됩니다.
