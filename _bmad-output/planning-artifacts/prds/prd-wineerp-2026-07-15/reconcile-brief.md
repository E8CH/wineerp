# Input Reconciliation — Brief → PRD (wineerp, 2026-07-15)

**목적:** 제품 브리프(brief.md + addendum.md)의 의미 있는 내용, 특히 정성적 요소(톤·의도·제약·리스크·전략)가 PRD(prd.md + addendum.md)로 이어졌는지, 아니면 조용히 누락됐는지 점검.

**결론(요약):** PRD는 브리프의 **기능적·구조적 내용**(문제 정의, 솔루션 흐름, 범위, POC/본제품 구분, 리스크→OQ 매핑, 성공지표, NFR, 톤앤매너 §12)을 매우 충실히 반영했다. 특히 리스크(바코드↔빈티지, 라벨 다양성, 매칭 기준 데이터), 성공기준(50% 단축·핵심 시퀀스 완주·빈티지 판정), 판매상태 제외 결정은 정확히 이관됐다. 그러나 브리프의 **전략적/정서적 정성 요소** 일부가 PRD에서 약화되거나 누락됐다. 아래 4건이 주요 갭.

## 잘 이관된 항목 (참고)

- **핵심 차별화 "명세서가 아니라 실물이 기준"** → PRD §1 Vision, JTBD, UJ에 명확히 반영.
- **와인 빈티지 = 별도 품목** 도메인 사실 → Glossary(와인 마스터/빈티지)에 1급 요소로 반영.
- **모바일 우선(관리자 포함)** → §2.2, §11 Platform, addendum의 idea.md 재조정 주석까지 반영.
- **판매/유통 상태(예약중·판매중·거래완료) 범위 제외** → §5 Non-Goals + 브리프 addendum 결정과 일치.
- **리스크 4종·오픈퀘스천** → §8 OQ-1~6, §13 Risk에 1:1 매핑(라벨 다양성=OQ-5 포함).
- **성공기준(POC/본제품)** → §7 SM-1~5 + 카운터메트릭으로 정교화.
- **톤앤매너(와인 분위기 + 한국인 정서)** → §12 Aesthetic and Tone에 그대로 반영.
- **성능/규모(하루 100병, 연 1,000종)** → §10 NFR 반영.

> 참고: PRD는 브리프에 없던 **"LLM(Gemini/OpenAI 클라우드) 라벨 모델명 추론"** 을 추가했다(브리프는 단순히 "라벨에서 읽은 기본 모델명"이라고만 함). 이는 누락이 아니라 **확장/구체화**이며, 그로 인한 지연·비용·외부 데이터 전송 리스크를 PRD가 OQ-6·§13·addendum으로 성실히 관리하고 있다. 다만 브리프 대비 스코프·비용을 키우는 결정이므로 OQ-1 범위 재협의와 연동해 인지할 것.

## 주요 갭 (브리프에 있으나 PRD에 충분히 반영되지 않음)

### 갭 1 — 전략적 해자(moat): "축적되는 고객 와인 DB = 전환비용"
- **브리프:** *What Makes This Different* + *Vision* — "정직한 해자. 기술 진입장벽은 크지 않고, 강점은 도메인 적합성·실행 속도·워크플로 밀착 커스터마이징. **POC 이후 축적되는 고객사 고유 와인 DB가 시간이 갈수록 전환비용을 만든다.**"
- **PRD:** 경쟁/전략 포지셔닝 프레이밍이 사실상 없음. 와인 마스터 DB는 §3에서 순수 **기능적 식별 단위**로만 다뤄지고, "축적될수록 자산·전환비용이 된다"는 전략적 의도가 사라짐.
- **영향:** PM/아키텍처가 DB 축적·데이터 소유·마이그레이션 저항성 같은 장기 가치를 놓칠 수 있음. 정성적 "왜 이 제품이 이기는가"가 실종.

### 갭 2 — 오너/회장의 정서적 성공 조건: "눈으로 빨리 보는 그림"
- **브리프 addendum(POC 상업조건):** "오너는 **'눈으로 빨리 보는 그림'** 중시", 담당자(황광현)가 회장께 보고·결재. [ASSUMPTION] 회장/오너 보고 시 그래프·리포트가 특히 중요.
- **PRD:** 그래프·엑셀을 **기능(FR-10/11)** 으로만 다룸. "시각적 임팩트가 POC 데모·결재의 정서적 핵심"이라는 의도는 SM-5에 희미하게만 남고, POC 데모 설계의 우선순위 근거로 명시되지 않음.
- **영향:** POC에서 리포트/그래프의 "데모 와우 요소"로서의 전략적 무게가 기능 백로그의 한 줄로 평탄화됨. 회장 결재라는 실제 성공 게이트가 흐려짐.

### 갭 3 — POC 상업/의사결정 맥락: 첫 외주·포트폴리오 할인·"POC 생략 직행" 가능성
- **브리프 addendum(POC 상업조건):** 고객사 **첫 외주**, 약 200만원, **포트폴리오 활용 동의 시 금액 일부 할인** 옵션(노출 범위 협의), **회장 판단에 따라 POC 없이 본과제 직행 가능성**.
- **PRD:** 예산(200만원)은 OQ-1에 언급되나, **"POC 생략 후 본과제 직행 가능성"**(→ 범위/일정 계획에 직접 영향), 포트폴리오 할인 옵션, 첫 외주라는 관계 맥락은 누락.
- **영향:** OQ-1(범위·비용 재협의)의 배경 정보가 빈약해짐. "직행" 시나리오는 MVP 범위 설계 분기를 좌우하는 제약인데 PRD에 없음.

### 갭 4 — 비전 확장성: "식별 까다로운 품목 전반의 표준 도구"
- **브리프 Vision:** "와인처럼 **식별이 까다롭고 실물 확인이 필수인 품목의 입고·재고**를, 명세서가 아니라 실물 기준으로 관리하는 **표준 도구**가 된다. …재고 가시성과 데이터 기반 분석까지 **깊이를 더해 간다**."
- **PRD §1 Vision:** 현재 와인 제품 설명으로 축소됨. **카테고리 확장(와인 → 식별난해 품목 일반)** 야망과 **분석 깊이 확대**라는 장기 방향이 빠짐.
- **영향:** 데이터 모델·아키텍처가 "와인 전용"으로 좁게 고정될 소지. 장기 확장 의도가 다운스트림에 전달되지 않음.

## 판정
브리프의 **기능·범위·리스크·톤** 이관은 우수(사실상 완전). 누락은 주로 **전략·정서·상업 맥락 등 정성 레이어**에 집중됨(갭 1~4). 이들은 FR을 바꾸진 않지만, POC 데모 우선순위·범위 재협의(OQ-1)·장기 데이터전략·비전 서술의 근거가 되는 요소이므로 PRD의 Vision(§1)·MVP/OQ(§6·§8) 서술 보강 시 반영을 권고한다.
